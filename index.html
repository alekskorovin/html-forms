Блочность элементов страницы

Все элементы на странице предаставляют собой прямоугольные области. Все видимые составляющие страницы можно представить себе как множество прямоугольников, содержащих внутри себя текст либо другое содержание.

Мы можем убедиться в этом добавив всем элементам (используя универсальный селектор) на странице внешнюю рамку CSS-свойством outline (это свойство аналогично свойству border за исключением того что внешняя рамка рисуется за пределами обычной рамки и нельзя применять border-radius)

Задание #1

  1. Скачать на рабочий стол архив "mobi2" и разархивировать его — в дальнейшем мы будем работать с этой заготовкой.
  2. Скачать на рабочий стол и открыть PSD-макет "mobi-home-desktop.psd" в Photoshop-e.
  3. Открыть в папке mobi-2 подпапку css и в редакторе кода открыть файл main.css и добавить стиль: 
    * {
    outline: solid 1px red;
}
    Открыть страницу index.html из папки mobi2 в браузере и убедиться в прямоугольности каждого элемента
  4. Открыть инспектор кода (DevTools) нажав F12 и нажать на иконку стрелочки (в левом верхнем углу панели).

    Посмотреть стили элементов в панели
  5. Убрать стили для обводки всех элементов: 
    * {
    outline: solid 1px red;
}

Использование блочности для декомпозиции компонентов дизайна

Мы можем рассмотреть компонент carousel понимая что каждый элемент этого компонента это прямоугольник.

Если идти от внешего края компонента к его внутреннему содержанию, можно выделить следующие элементы:

    Он содержит размытый фон и лесенку внизу, так же он тянется на всю ширину экрана.

  • Сами слайды как правило это элементы списка. Список будет располагаться внутри контейнера. Мы будем использовать распространённый flexslider в котором заложена следующая структура HTML:

    Вместо изображений у нас будет HTML
  • Сам слайд содержащий в себе контейнеры для изображения и контента
  • Контейнер для размещения изображения. Для примера у нас будет 3 разных слайда с разными изображениями.

    Внутри этого контейнера будет находиться полупрозрачное изображение
  • Контейнер для размещения контента. Для примера у нас будет 3 разных слайда с разным текстом.
  • Подзаголовок слайда
  • Параграф текста слайда
  • Ссылка на страницу регистрации (sign up)
  • Элементы управления каруселью — стрелки влево и вправо, по нажатию на эти стрелки предполагается:
    • влево — идёт переход к предыдущему элементу (previos)
    • вправо — идёт переход к следующем элементу (next)

Создание CSS для компонента carousel

Общий контейнер карусели с фоном

Давайте зададим стили для общего контейнера карусели с фоном. Фон включает в себя два изображения:

  • Размытое изображение, растянутое на всю ширину экрана carousel-background.jpg
  • Узор в виде треугольничка battlement.png, который повторяется по горизонтали внизу над размытым изображением.

Здесь можно использовать одну запись включающую себя оба фона с разными параметрами.

Так же мы можем сразу добавить ограничение по высоте для компонента (max-height) и обрезку его содержимого (overflow: hidden; — обрезает содержимое если задан размер), чтобы он не "прыгал" при загрузке (элементы списка по умолчанию идут один под другим и только после применения скрипта выстраиваются в ряд).

Задание #13

  1. Открыть таблицу стилей main.css в редакторе кода.
  2. Добавить пустую строку между комментариями: 
    /* =Carousel */

/* Carousel */
  3. Добавить вместо пустой строки в таблицу стилей main.css правила для класса carousel. 
    /* =Carousel */
.carousel {
    background: url(../img/battlement.png) repeat-x 0 100%, url(../img/carousel-background.jpg) no-repeat 0 0 fixed;
    background-size: 12px 8px, 100% 100%;
    overflow: hidden;
    max-height: 48.8rem;
}
/* Carousel */
  4. Обновить страницу в браузере и попробовать поменять её ширину. Попробовать прокрутить страницу вниз-вверх.

Весь текст в карусели белый, поэтому можно сразу добавить цвет для текста и для ссылок.

Задание #14

    /* =Carousel */

/* Carousel */
  2. Обновить страницу в браузере и убедиться что цвет поменялся.

Ссылка на страницу регистрации

Эта ссылка (селектор .carousel-slides .carousel-button выглядит как кнопка, поэтому зададим следующие стили:

  • размер шрифта
  • высоту строки (чтобы кнопка была не сильно большой по вертикали)
  • уберём подчёркивание
  • сделаем элемент строчно-блочным, чтобы задать внешний отступ
  • добавим верхний внешний отступ (чтобы кнопка была ниже параграфа)
  • зададим внутренние оступы (чтобы синий фон был на расстоянии от текста, слева он будет больше чтобы текст не налазил на значок, который мы сделаем как псевдо-элемент и спозиционируем его абсолютно над фоном и текстом)
  • зададим цвет фона
  • добавим радиус скругления
  • зададим относительное позиционирование (чтобы раместить значок на абсолютном позиционировании относительно кнопки)

Элементы управления каруселью — стрелки влево и вправо

Это две стрелки, которые вынесены за пределы блока со слайдами. Обе стрелки находятся на одном уровне по высоте и абсолютно спозиционированны, у них одинаковый размер. Таким образом задав для них один общий класс carousel-control мы можем вынести в него все общие свойства.

  2. Обновить страницу в браузере и убедиться что обе стрелки на месте.

Теперь можно добавить JavaScript который оживит нашу карусель. Для начала мы подключим библиотеку jQuery и Flexslider.

Задание #27

  1. Открыть index.html и добавить следующий код перед закрывающим тегом body:

<script src="http://flexslider.woothemes.com/js/jquery.flexslider.js"></script>

Для того чтобы карусель (слайдер) Flexslider заработал, в HTML надо добавить пару CSS-классов:

  • для блока с классом carousel-inner добавить дополнительный класс flexslider
  • для блока с классом carousel-slides добавить дополнительный класс slides

Задание #28

  1. Открыть index.html и добавить классы как описано выше: 
    <div class="carousel-inner flexslider">
    <ul class="carousel-slides slides">

Теперь остаётся добавить скрипт который запустит карусель с нужными нам параметрами:

  • типа анимации "slide" — элементы "выезжают" справа налево
  • выключим стандартную навигацию влево-вправо
  • выключим навигацию по каждому слайду (кружочки внизу)
  • добавим управление на собственные кнопки с классами carousel-prev и carousel-next

Задание #29

  1. В index.html после добавленых скриптов и перед закрывающим тегом body добавить скрипт "оживляющий" карусель: 
    <script>
    (function() {
        $(window).load(function() {
            var $flexslider = $('.flexslider');
            $flexslider.flexslider({
                animation: "slide",
                directionNav: false,
                controlNav: false
            });
            $('.carousel-prev').click(function(){
                $flexslider.flexslider('prev');
            });
            $('.carousel-next').click(function(){
                $flexslider.flexslider('next');
            });
        });
    })();
</script>
